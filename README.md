Web application for LibreLock, a secure, self-hosted password manager. Built with [Vue](https://vuejs.org/) and [Vite](https://vitejs.dev/).

## Get started

Refer to _Get started_ section in this [README](https://github.com/LibreLock/) for one command setup of the entire application. Keep reading if you wish to run the frontend separately.

The preferable way to run LibreLock Web is via Docker Compose:

```bash
docker compose up -d
```

To run without Docker use:

```bash
npm install

# Dev server
npm run dev

# Build for production
npm run build
npm run preview
```
